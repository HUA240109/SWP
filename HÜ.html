Einige interessante Elemente in der HTML

  1. details : Macht einen aufklappbaren Bereich, wie bei „Mehr anzeigen“.
  2. summary : Die Überschrift für den aufklappbaren Bereich.
  3. canvas : Eine Zeichenfläche, auf der man mit JavaScript malen kann (z.B. für Spiele oder Animationen).
  4. dialog : Ein kleines Fenster, das sich öffnen lässt; wie ein Pop-up.
  5. mark : Hebt Text hervor, z.B. gelb hinterlegt wie bei Suchtreffern.
  6. output : Zeigt ein Ergebnis an, z.B. nach einer Berechnung.
  7. progress : Zeigt, wie weit etwas ist; wie ein Ladebalken.
  8. meter : Zeigt einen Wert in einem Bereich, z.B. eine Bewertung von 1 bis 10.
  9. time : Zeigt ein Datum oder eine Uhrzeit, auch für Maschinen lesbar.
  10. abbr : Kennzeichnet eine Abkürzung, z.B. „HTML“ mit Erklärung.
  11. kbd : Zeigt Tasten, die man drücken soll, z.B. „Strg + C“.
  12. code : Hebt Programmiercode hervor.
  13. samp : Zeigt eine Beispiel-Ausgabe von einem Programm.
  14. ruby : Für asiatische Schriftzeichen mit Aussprachehilfe.
  15. wbr : Gibt dem Browser eine Stelle, wo er den Text umbrechen darf.
  16. fieldset : Gruppiert Felder in einem Formular, z.B. „Persönliche Daten“.
  17. legend : Die Überschrift für einen Formularbereich.
  18. picture : Zeigt je nach Bildschirmgröße ein anderes Bild.
  19. source : Gibt verschiedene Bild- oder Tonformate zur Auswahl.
